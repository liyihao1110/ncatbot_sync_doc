# 👋 这是NcatBot的官方文档

NcatBot是一个使用Python语言编写的QQ框架项目，它是基于NapCat.QQ创建的，所以你需要知道，只有处于NapCat.QQ环境下，你才可以正常使用NcatBot哦\~

如果你想创建一个自己的QQ机器人，可以尝试使用我们的项目。

NcatBot是一个开源的基于Napcat.QQ开发的PythonSDK，使用python调用QQ。

使用简单的代码，你就可以完成一个能够处理所有信息的QQ机器人。

你还可以使用别人编写的插件！！！
